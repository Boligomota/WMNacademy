# Source Policy

Use verified source monitoring, not uncontrolled scraping.

## Allowed

- Official platform docs
- Official blogs
- Government institutions
- Universities
- Public advisories
- Credible media
- Public reports

## Not allowed

- Private Facebook groups
- Login-protected pages
- Paywalls
- Restricted pages
- Generic SEO blogs
- Viral claims without sources
- Full article copying

## Required source fields

- name
- url
- date
- type
- trust reason
- claim
- practical relevance
