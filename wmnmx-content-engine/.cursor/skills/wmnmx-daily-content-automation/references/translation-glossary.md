# Translation Glossary

Use original digital term + Spanish explanation.

- Phishing: una estafa donde alguien se hace pasar por una institución o marca para robarte datos.
- Ghosting: cuando alguien desaparece sin responder; en negocio, puede ser un prospecto que pregunta mucho y nunca concreta.
- Love bombing: cuando alguien te llena de atención, promesas o halagos para manipularte.
- DM: mensaje directo.
- Prompt: la instrucción que le das a una IA para que te ayude con una tarea.
- Deepfake: video, imagen o audio falso hecho con IA.
- 2FA: verificación en dos pasos.
- Passkey: llave digital para entrar sin contraseña tradicional.
- Malware: software malicioso.
- Scam: estafa.
- Bot: cuenta o sistema automatizado.
- Chatbot: asistente automático que responde mensajes.
- Workflow: secuencia de pasos para hacer una tarea.
- No-code: herramientas para crear procesos o páginas sin programar.
- Low-code: herramientas con muy poco código.
- Social commerce: vender desde redes sociales.
- E-commerce: comercio electrónico.
- Marketplace: espacio digital donde varias personas o marcas venden.
- Lead: persona interesada que podría convertirse en cliente.
- Checkout: paso final para pagar.
- Link-in-bio: link en la biografía de una red social.
