# Quality Gate

Every post is scored out of 30.

| Criterion | Score |
|---|---:|
| Source trust | 0-5 |
| Clear date | 0-5 |
| Clarity | 0-5 |
| Immediate usefulness | 0-5 |
| Relevance for women 40+ | 0-5 |
| Conversation potential | 0-5 |

## Status

- 24/30 or more: ready for human review.
- 18-23: revise.
- 17 or less: reject.

## Auto-reject

Reject if:

- source is not trustworthy
- source has no date and topic requires recency
- content is fear-based
- content is too technical
- content has no 10-minute action
- content sounds like guru marketing
