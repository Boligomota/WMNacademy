# Prompt Patterns

## Simple prompt structure

```txt
Actúa como [rol].
Ayúdame a [tarea].
Contexto: [contexto].
Tono: [tono].
Formato de salida: [formato].
```

## Example for WMNMX

```txt
Actúa como una asistente de comunicación clara.
Ayúdame a escribir un mensaje de WhatsApp para una clienta que preguntó por mis servicios.
Contexto: vendo productos hechos a mano y quiero sonar amable, profesional y no insistente.
Formato: 3 opciones cortas.
```
