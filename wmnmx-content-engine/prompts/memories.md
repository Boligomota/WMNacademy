# Suggested Cursor Memories for WMNMX Automation

Use only stable, high-level memory notes. Do not store full strategy documents as Memories.

## Memory 1 — Positioning

WMNMX is a private Facebook group for women 40+ in Mexico. It helps women adopt technology with confidence, safety, productivity, professional reinvention, AI literacy, and social commerce. The group is a school-community, not a news feed.

## Memory 2 — Editorial operating model

WMNMX automation should work as an editorial desk: source intake, signal ranking, verification, novelty score, audience fit, Spanish editorial translation, human review, and manual Facebook scheduling. It must not behave like an autoposter or a scraper.

## Memory 3 — Source policy

Use verified public sources only. Prefer RSS feeds, public APIs, official documentation, official blogs, government institutions, universities, public reports, platform help centers, and reputable media that cite primary sources. Do not use private Facebook groups, login-protected pages, paywalls, restricted pages, viral posts without sources, undated content for current topics, generic SEO blogs, content farms, or copied full articles.

## Memory 4 — Translation policy

Final outputs must be in Spanish. Keep key digital terms in English when the audience needs to recognize them online: phishing, ghosting, love bombing, DM, prompt, deepfake, 2FA, passkey, malware, scam, bot, chatbot, workflow, no-code, low-code, social commerce, e-commerce. Explain each term simply in Spanish.

## Memory 5 — Output standard

Every daily run must create `/content/daily/YYYY-MM-DD/` with ranked signals, research summary, posts ready, visual briefs, sources used, sources rejected, quality score, and editorial notes.

## Memory 6 — Novelty and audience fit

Do not force three posts daily. One excellent post is better than three generic posts. Always check whether the topic was recently used and whether it responds to a real need, question, fear, or goal from women 40+ in Mexico.

## Memory 7 — Facebook boundary

Do not connect to Facebook or publish automatically in phase 1. For private group insights, use only human-provided notes, manual summaries, or permitted links. Never scrape private group content.
